Bank Loan Analytics Dashboard | Power BI & SQL
Project Overview

This project analyzes bank loan performance using SQL and Power BI. The dashboard provides insights into loan portfolio performance, repayment trends, customer behavior, and risk metrics.

Tools Used
Power BI
SQL
DAX
Power Query
Microsoft Excel
Key KPIs
Total Loan Amount
Total Amount Received
Average Interest Rate
Fully Paid Loans
Charged-Off Loans
Current Loans
Features
Year-wise Loan Analysis
State-wise Loan Performance
Grade-wise Loan Analysis
Verification Status Analysis
Home Ownership Analysis
Loan Status Tracking
Business Impact

The dashboard helps stakeholders monitor loan performance, identify repayment trends, assess risk, and make data-driven decisions.
